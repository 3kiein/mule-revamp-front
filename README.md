## 프로젝트 소개

### 우리만의 자유로운 99's Mule

1999년, 밴드 '뮬'의 홈페이지로 시작된 뮤지션 커뮤니티인 Mule을 MZ만의 방식으로 개편해 보았습니다.

### 조금 더 보기 편한 Mule

1. 최신 글을 홈에서 더욱 빠르게 볼 수 있습니다.
2. 뮬에서 가장 인기 많은 게시판, 중고장터를 위한 캐러셀 구현
3. 더욱 직관적으로 변한 카테고리 분류

### 그럼에도, 본질은 잊지 않아야 하니까
